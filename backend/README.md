# Stravo Backend

Node.js/Express API for rural path discovery, activity recording, segment scoring, and feedback aggregation.

## Structure

- `src/config/`: Environment and database setup
- `src/controllers/`: Route handlers
- `src/models/`: Sequelize models and schema
- `src/routes/`: API route definitions
- `src/services/`: Business logic for activities and path analysis
- `src/middleware/`: Auth and error handling
- `src/workers/`: Background aggregation jobs
