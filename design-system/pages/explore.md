# Explore Page Overrides

Inherits from `MASTER.md`.

## Layout

- **Mobile**: full-screen map + segment polylines + bottom sheet on tap
- **Web**: sidebar (filters + list) + full-width map

## Segment colors

- Score ≥ 60%: forest green (`#2D6A4F`)
- Unrated: slate gray (`#94A3B8`)
- Score < 40%: red tint (`#DC2626`)

## Feedback

- Thumbs up / thumbs down in bottom sheet
- ≤ 2 taps to submit
